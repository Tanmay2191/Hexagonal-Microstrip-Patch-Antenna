# 📡 2.4GHz Hexagonal Patch Antenna

This repository contains the electromagnetic simulation and design files for a 2.4GHz hexagonal patch antenna. The antenna was modeled and simulated using ANSYS HFSS (Electronics Desktop) and is optimized for the 2.4GHz ISM band (commonly used for Wi-Fi and Bluetooth applications).

## ⚙️ Design Specifications
* **Operating Frequency:** 2.4 GHz
* **Antenna Geometry:** Hexagonal Patch
* **Simulation Software:** ANSYS HFSS (High-Frequency Structure Simulator)
* **Application:** ISM Band / Wireless Communications

### How to use the files:
1. Download the `.aedt` file from the link above.
2. Open the file in ANSYS Electronics Desktop.
3. To view the field overlays and performance graphs, navigate to the results tab or select "Analyze All" to regenerate the solution data on your local machine.
